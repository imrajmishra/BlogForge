# BlogForge Features

## 🚀 Overview

BlogForge is a modern social blogging platform that combines content creation, user profiles, short-form reels, and personalized settings into a seamless experience.

---

## 👤 User Profile

* View and manage personal profile information.
* Update display name, bio, and profile avatar.
* Personalized profile page with user-specific content.
* Responsive profile layout for desktop and mobile devices.

---

## 🎬 Reels

* Browse short-form content in an engaging feed.
* Smooth navigation between reels.
* Optimized UI for content discovery.
* Responsive viewing experience across devices.

---

## ⚙️ Settings

### Account Settings

* Update profile information.
* Change display name and bio.
* Upload and preview profile avatar.

### Appearance Settings

* Customize accent colors.
* Toggle glassmorphism effects.
* Personalized visual preferences.

### Security Settings

* Change account password.
* Basic credential management.
* User-friendly validation and feedback messages.

---

## 🧩 Shared Components

* Reusable UI components for consistency.
* Modular component architecture.
* Improved maintainability and scalability.
* Shared styling and interaction patterns.

---

## ✨ User Experience

* Modern and responsive interface.
* Smooth animations powered by Framer Motion.
* Real-time notifications and feedback.
* Mobile-friendly design.
* Clean and intuitive navigation.

* Built with Next.js.
* Component-driven architecture.

* Reusable and scalable code structure.
* Optimized developer experience and maintainability.
